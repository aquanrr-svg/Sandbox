# Sandbox App

Minimal React sandbox app (Vite).

Quick start:

1. Install dependencies
   ```bash
   npm install
   # or
   yarn
   ```
2. Start dev server
   ```bash
   npm run dev
   # or
   yarn dev
   ```
3. Build for production
   ```bash
   npm run build
   ```
4. Preview build
   ```bash
   npm run preview
   ```

Notes
- The app loads `sample-module.json` from the site root. Replace the loader with a backend endpoint or a modules folder as needed.
